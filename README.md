# My Digital Workshop: A Claymorphic Portfolio

Hey there! Welcome to the source code for my personal corner of the internet. This isn't just a standard portfolio; it's a digital space I've built from the ground up to reflect my journey as a developer and my love for creating unique, tactile web experiences.

Think of it as my personal workshop, where I blend the logic of embedded systems with the art of frontend development. The entire site is wrapped in a "claymorphic" design, giving it a soft, 3D, and friendly feel that I hope makes you want to click around and explore.

## ‚ú® Core Technologies

This project was brought to life with a modern, lightweight tech stack:

-   **[Vite](https://vitejs.dev/)**: For a lightning-fast development experience.
-   **[React](https://react.dev/)**: The heart of the user interface.
-   **[TypeScript](https://www.typescriptlang.org/)**: To keep the code clean, robust, and scalable.
-   **[Tailwind CSS](https://tailwindcss.com/)**: For crafting the custom, claymorphic UI with utility classes.

## üõ†Ô∏è Getting Started

Want to run the project on your own machine? It's easy!

1.  **Clone the repository:**
    ```bash
    git clone <your-repository-url>
    ```

2.  **Navigate to the project directory and install the dependencies:**
    ```bash
    cd <project-directory>
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  **Open your browser** and visit [http://localhost:5173](http://localhost:5173) (or whatever port Vite assigns) to see the magic happen!

### Other Available Scripts

-   **`npm run build`**: To create a production-ready build of the site.
-   **`npm run preview`**: To preview the production build locally.
